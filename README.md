## 실행방법

로컬 폴더에 해당 레포지토리를 클론한 후, 터미널 창에서 아래 코드를 차례대로 실행한 다음, localhost/product/로 접속하면 나옵니다.
```python
python manage.py createsuperuser
python manage.py runserver
```
